# 关于 OneLook

**OneLook（一目）** 是一款专注于沉浸式创作的开源思维导图工具。

## 设计理念

- **极简至上**：摒弃繁杂的 UI 干扰，让您专注于思考本身
- **本地优先**：数据完全存储于本地，隐私无忧
- **开箱即用**：无需登录注册，打开即可创作

## 技术栈

| 类别 | 技术 |
|------|------|
| 框架 | Vue 3 + TypeScript |
| 构建 | Vite 6 |
| 状态 | Pinia |
| 渲染 | SVG + D3.js |
| 存储 | IndexedDB (Dexie.js) |
| 文档 | VitePress |
