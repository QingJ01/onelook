# 文件与数据管理

## 📂 多标签页管理
OneLook 采用了类似浏览器的多标签页设计：
- **同时编辑**：顶部的标签栏允许您同时打开多个导图文件。
- **快速切换**：点击标签即可在不同导图间即时切换，无需重新加载。
- **状态同步**：标签栏会实时反映当前文档的名称和选中状态。

## 💾 本地数据存储
我们非常重视您的隐私和数据主权。
- **IndexedDB 技术**：OneLook 使用先进的浏览器内置数据库 (IndexedDB) 存储所有数据。
- **无需联网**：所有内容都保存在您的电脑上，**不会**上传到任何云端服务器。这意味着：
  - 即使断网也能完整使用。
  - 您的数据完全私有，没有任何第三方（包括我们）可以访问。
- **实时保存**：无需手动点击保存按钮，您的每一次修改都会毫秒级自动持久化到本地数据库。

## 📤 导入与导出

### 导出
点击右上角的「导出」按钮，支持多种格式：

#### OneLook 格式
- **OneLook 文件 (.olook)**：原生格式，完整保留所有样式、布局和状态，适合备份或在不同设备间迁移。

#### 通用格式
- **JSON (.json)**：通用数据格式，可用于数据交换。
- **Markdown (.md)**：导出为 Markdown 列表，方便转换成文章大纲。

#### 图片格式
- **高清图片 (.png)**：适合分享给没有 OneLook 的朋友，或插入到文档中。
- **矢量图 (.svg)**：无限放大的矢量格式，适合用于印刷或高分屏展示。

#### 第三方格式
- **XMind (.xmind)**：兼容 XMind 8+ 格式，支持节点样式、备注、标签、优先级、进度等。
- **FreeMind (.mm)**：兼容 FreeMind XML 格式，支持节点样式、备注、链接、图标等。
- **OPML (.opml)**：大纲处理标记语言，兼容多种大纲工具和 RSS 阅读器。

### 导入
点击工具栏的「导入」按钮，支持以下格式：
- **OneLook (.olook)**：原生格式，完整还原所有内容。
- **XMind (.xmind)**：导入 XMind 思维导图，自动转换节点样式和属性。
- **FreeMind (.mm)**：导入 FreeMind 思维导图，保留节点结构和样式。
- **OPML (.opml)**：导入 OPML 大纲文件。
- **JSON (.json)**：导入 JSON 格式的思维导图数据。
- **Markdown (.md)**：从 Markdown 大纲创建思维导图。

::: tip 智能识别
导入功能会自动识别文件格式，无需手动选择导入类型。
:::

::: warning 注意
由于浏览器的安全限制，清除浏览器缓存可能会导致本地保存的数据丢失。强烈建议定期导出重要的导图为 `.olook` 文件进行备份。
:::
