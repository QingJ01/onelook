# 更新日志 (Changelog)

所有项目的重要更新都将记录在此页面。

## v0.2.2 (2026-02-07)

### ✨ 新特性
- **节点图片展示增强**：
  - 支持节点图片与节点文本同时展示，文本默认位于图片下方。
  - 新增图片宽度调节后，节点内容可实时重排。
- **图文布局自适应**：
  - 节点内容内边距、图文间距、边框留白统一改为自适应计算。
  - 图片展示高度支持按真实宽高比估算，减少文本被裁切或挤压的问题。

### 🛠️ 优化与修复
- 优化 Markdown 渲染容器在图文混排场景下的布局稳定性。

## v0.2.1 (2026-02-02)

### ✨ 新特性
- **知名思维导图格式兼容**：
  - 新增 **XMind (.xmind)** 格式导入/导出支持，兼容 XMind 8+ 版本。
  - 新增 **FreeMind (.mm)** 格式导入/导出支持，兼容 FreeMind XML 格式。
  - 新增 **OPML (.opml)** 格式导入/导出支持，兼容大纲工具和 RSS 阅读器。
  - 支持节点样式、备注、链接、标签、优先级、进度等属性的完整转换。
  - 智能导入功能：自动识别文件格式，无需手动选择导入类型。

### 🛠️ 优化与改进
- **导出菜单重构**：
  - 导出菜单按格式类型分组显示（OneLook 格式、通用格式、图片格式、第三方格式）。
  - 添加导出成功/失败的 Toast 通知提示。
- **格式转换器架构**：
  - 新增模块化的格式转换器系统 (`src/services/converters/`)。
  - 统一的导入导出接口，便于后续扩展更多格式。

### 📦 依赖更新
- 新增 `jszip@3.10.1` 用于 XMind ZIP 文件处理。

## v0.2.0 (2026-02-01)

### ✨ 新特性
- **演说模式 (Pitch Mode)**：
  - 沉浸式演示体验，无需导出 PPT，直接在浏览器中进行汇报。
  - 支持 **智能运镜**，自动平移缩放并聚焦当前节点。
  - 提供 **HUD 控制台**，包含翻页导航、激光笔和计时器功能。
  - 支持 **逐步演示** 和 **全图浏览** 模式切换。
  - **Light/Dark 主题适配**：演示模式自动跟随应用主题。
- **彩虹分支 (Rainbow Branch)**：
  - 设置中新增一键开关，为主分支及其子节点赋予鲜明的彩虹色彩，提升阅读体验。
  - 演说模式下节点边框自动继承该分支颜色。

### 🛠️ 优化与修复
- **动画优化**：为演说模式的相机移动添加了平滑的贝塞尔曲线过渡。
- **样式修复**：修复了 SVG `foreignObject` 中 Markdown 内容的样式穿透问题。
- **ViewBox 修复**：修复了全屏模式下 viewBox 计算不准确导致内容裁切的问题。
- **交互优化**：演说模式下点击空白处即可翻页，键盘支持 ESC 退出。

## v0.1.0 (2026-01-24)

### 🚀 新特性
- **关于弹窗**：设置菜单中新增“关于 OneLook”弹窗，展示项目信息、社会链接及版权。
- **沟通渠道**：正式开启官方 QQ 交流群 (1081533635)，支持一键加入与扫码。

### 🛠️ 体验优化
- **键盘冲突优化**：引入 IME (输入法) 状态检测，修复中文输入过程中误触发快捷键的问题。
- **导出反馈**：图片导出操作现在会显示 Loading 状态及成功/失败的 Toast 提醒。
- **导航同步**：优化 TabBar 与 文件管理器的同步逻辑，确保多标签页切换流程。

### 🛡️ 安全与质量
- **数据校验**：强化 JSON 导入逻辑，增加 Schema 校验及嵌套深度限制，防止恶意数据导致应用崩溃。
- **错误处理**：使用自定义 Toast 组件替换原生的 `alert()`，提供更友好的异常提示。
- **构建优化**：修复 TypeScript 配置冲突，补全 Node.js 类型定义，确保 Vercel 环境一键构建通过。
