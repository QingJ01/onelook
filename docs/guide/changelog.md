# 更新日志 (Changelog)

所有项目的重要更新都将记录在此页面。

## v0.2.0 (2026-02-01)

### ✨ 新特性
- **演说模式 (Pitch Mode)**：
  - 沉浸式演示体验，无需导出 PPT，直接在浏览器中进行汇报。
  - 支持 **智能运镜**，自动平移缩放并聚焦当前节点。
  - 提供 **HUD 控制台**，包含翻页导航、激光笔和计时器功能。
  - 支持 **逐步演示** 和 **全图浏览** 模式切换。
  - **Light/Dark 主题适配**：演示模式自动跟随应用主题。
- **彩虹分支 (Rainbow Branch)**：
  - 设置中新增一键开关，为主分支及其子节点赋予鲜明的彩虹色彩，提升阅读体验。
  - 演说模式下节点边框自动继承该分支颜色。

### 🛠️ 优化与修复
- **动画优化**：为演说模式的相机移动添加了平滑的贝塞尔曲线过渡。
- **样式修复**：修复了 SVG `foreignObject` 中 Markdown 内容的样式穿透问题。
- **ViewBox 修复**：修复了全屏模式下 viewBox 计算不准确导致内容裁切的问题。
- **交互优化**：演说模式下点击空白处即可翻页，键盘支持 ESC 退出。

## v0.1.0 (2026-01-24)

### 🚀 新特性
- **关于弹窗**：设置菜单中新增“关于 OneLook”弹窗，展示项目信息、社会链接及版权。
- **沟通渠道**：正式开启官方 QQ 交流群 (1081533635)，支持一键加入与扫码。

### 🛠️ 体验优化
- **键盘冲突优化**：引入 IME (输入法) 状态检测，修复中文输入过程中误触发快捷键的问题。
- **导出反馈**：图片导出操作现在会显示 Loading 状态及成功/失败的 Toast 提醒。
- **导航同步**：优化 TabBar 与 文件管理器的同步逻辑，确保多标签页切换流程。

### 🛡️ 安全与质量
- **数据校验**：强化 JSON 导入逻辑，增加 Schema 校验及嵌套深度限制，防止恶意数据导致应用崩溃。
- **错误处理**：使用自定义 Toast 组件替换原生的 `alert()`，提供更友好的异常提示。
- **构建优化**：修复 TypeScript 配置冲突，补全 Node.js 类型定义，确保 Vercel 环境一键构建通过。
