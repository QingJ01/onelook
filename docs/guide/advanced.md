# 高级技巧

掌握这些进阶技巧，将助您从“会用”进阶为“精通”。

## 📑 大纲视图编辑

当您的导图结构非常复杂时，图形化视图可能会显得拥挤。OneLook 提供了同步的**大纲视图**，让您像编辑 Word 文档一样处理导图。

### 开启大纲
点击左上角的 **列表图标** (List Icon)，侧边栏会展开当前导图的大纲树。

### 大纲操作
- **双击编辑**：直接双击大纲中的任意文字，即可进行即时修改。
- **点击定位**：点击大纲条目，右侧画布会自动平移并选中对应的图形节点。
- **实时同步**：在大纲中的所有修改（文本、增删）都会毫秒级同步到图形视图，反之亦然。

## 🔍 全局搜索与替换

不再为找不到某个知识点而发愁。

### 唤起搜索
按下 `Ctrl + F` (Mac: `Cmd + F`)，屏幕左上角会弹出搜索面板。

### 功能详解
- **快速查找**：输入关键词，OneLook 会高亮所有匹配的节点，并自动跳转到第一个结果。
- **查找下一个**：按 `Enter` 键或点击下箭头 `↓`，跳转到下一个匹配项。
- **批量替换**：
  1. 点击搜索框右侧的箭头展开 **替换面板**。
  2. 输入“查找内容”和“替换为”。
  3. 点击 **替换** (Replace) 修改当前选中项。
  4. 点击 **全部替换** (Replace All) 一键修改所有匹配内容。

## 🧘 禅模式工作流

如果您经常处理大图，建议把禅模式当作“专注滤镜”使用。

### 推荐切换策略
1. **先用 Lite 整理结构**：保留必要工具，快速增删改节点。
2. **再切到 Deep 深度写作**：进入纯净画布，连续完善内容。
3. **按任务切聚焦模式**：
   - 梳理一个主题链路时用 `Path`
   - 深挖一个章节时用 `Branch`
   - 讲解全图时用 `Highlight`

### 临时查看上下文
- 在 `Branch / Path` 模式下，按住 `Alt` 可短暂查看全图，再松开回到聚焦状态。

### Esc 的优先级
- `Esc` 会先退出当前临时状态（如菜单、选择态），再回到普通编辑视图。
- 若希望快速离开禅模式，建议先清空当前选中节点后再按 `Esc`。

## 🏗️ 深度布局与拖拽逻辑

OneLook 的拖拽系统非常智能，理解它的逻辑能让您更得心应手。

### 拖拽的两种模式
1. **改变父子关系 (Restructure)**
   - **操作**：拖拽一个节点 A，将其悬停在节点 B 上方（节点 B 会变色）。
   - **结果**：节点 A 成为节点 B 的子节点。
   
2. **改变兄弟顺序 (Reorder)**
   - **操作**：拖拽一个节点 A，将其悬停在节点 B 的**上半部分**或**下半部分**（会出现一条蓝色指示线）。
   - **结果**：节点 A 移动到节点 B 的前面或后面，成为同级兄弟。

### 自由布局微调
- 默认情况下，所有节点受自动布局算法控制。
- 如果您**按住拖动**某节点到空白区域，该节点会进入**自由定位模式**，位置不再随自动布局变化。
- **恢复自动**：只需再次拖动该节点，使其吸附回父节点即可。

## 🎨 样式进阶：节点与连线

在右侧属性面板中，您可以定制更多细节：

### 节点形状
除了默认的圆角矩形，您还可以在属性面板的「样式」标签下选择：
- **矩形**：严谨、正式。
- **椭圆**：柔和、发散。
- **圆角矩形**：通用、现代。

### 连线风格
在全局设置（右上角 ⚙️）中，您可以统一修改连线风格：
- **曲线 (Curve)**：默认风格，优雅流畅。
- **直线 (Straight)**：简洁干练，适合树形图。
- **折线 (Polyline)**：硬朗清晰，适合组织架构图或流程图。

::: tip 混合搭配
不同的布局（如鱼骨图、组织图）配合不同的连线风格，会产生意想不到的专业效果。
:::
