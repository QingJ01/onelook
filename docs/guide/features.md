# 核心功能

## 📝 强大的内容编辑

OneLook 不仅仅能写纯文本，还支持丰富的排版：

### Markdown 语法支持
- **加粗**：输入 `**文字**`
- *斜体*：输入 `*文字*`
- ~~删除线~~：输入 `~~文字~~`
- `代码片段`：输入 `` `代码` ``
- [链接](https://example.com)：输入 `[链接文字](网址)` (链接会在新标签页安全打开)

### 数学公式 (LaTeX)
学术利器！直接输入 LaTeX 语法即可渲染精美的数学公式。

**常见示例**：
- **行内公式**：输入 `$E=mc^2$` 显示为 $E=mc^2$
- **分数**：输入 `$\frac{a}{b}$` 显示为 $\frac{a}{b}$
- **根号**：输入 `$\sqrt{x}$` 显示为 $\sqrt{x}$
- **求和**：输入 `$$\sum_{i=1}^n i$$` 显示为块级求和公式

::: info 预览机制
编辑节点时编辑器会显示原始 LaTeX 代码，以便您修改。当您选中其他节点或按 `Esc` 退出编辑后，系统会自动调用渲染引擎显示最终效果。
:::

## 🎨 个性化样式

让您的导图独一无二：

### 一键换肤与主题
点击右上角的设置图标（⚙️），您可以立即切换预设主题。主题不仅改变背景色，还会智能调整节点的边框、文字颜色和连线颜色，确保在任何背景下都有最佳的对比度。
- 🌞 **亮色 (Light)**：经典白色背景，高对比度，适合办公与打印。
- 🌙 **暗色 (Dark)**：深空灰背景，护眼模式，适合夜间创作。
- 🍃 **清新 (Fresh)**：淡雅配色，适合轻松的头脑风暴。

### 布局切换
支持多种经典结构，适应不同场景：
- **思维导图 (Mind Map)**：中心发散，适合创意发散。
- **树形结构 (Tree)**：自顶向下或自左向右，适合层级分明的分类。
- **鱼骨图 (Fishbone)**：因果分析神器，问题溯源。
- **组织架构图 (Org Chart)**：垂直层级，适合人员管理。

### 节点深度定制
选中任意节点，在右侧属性面板中，您可以打破全局主题的限制，进行个性化定制：
- **字体样式**：调整字号大小、加粗 (`Font Weight`)，突出重点节点。
- **节点形状**：
  - **矩形**：适合长文本。
  - **圆角矩形**：默认样式，柔和。
  - **椭圆**：适合短标签或强调。
- **边框与填充**：自定义边框颜色、宽度和背景填充色。

## 🚀 高效交互进阶

像专业人士一样飞速操作：

### 多选节点
- **Ctrl + 点击**：按住 Ctrl 键，依次点击想要选中的多个节点。
- **Shift + 拖拽**：按住 Shift 键，在画布空白处按住鼠标左键拖拽出一个矩形框，框选多个节点。

### 批量处理
选中多个节点后，您可以一次性执行以下操作：
- **批量删除**：按 `Delete` 键。
- **批量移动**：拖拽任意一个选中节点，所有选中的节点都会跟随移动到新的父节点下。

### 自由/自动布局混合模式
- **自动吸附**：默认情况下，节点会自动排列整齐。
- **自由定位**：如果您想微调位置，可以直接拖拽节点到任意空白处，它会固定在您放置的位置。
- **恢复自动**：如果想让自由节点重新变回自动排列，只需稍微触碰一下它的父节点即可吸附回去。

## 🧘 禅模式与聚焦

禅模式用于减少干扰，让注意力集中在当前思路。

### 开启方式
- 点击工具栏中的 **眼睛图标** 进入禅模式。
- 点击同一按钮或使用浮动条中的 **退出** 按钮可离开禅模式。

### 双层级沉浸
- **Lite**：保留精简工具栏，仅显示高频操作。
- **Deep**：隐藏标签栏、状态栏与完整工具栏，进入更纯粹的编辑视图。

### 三种聚焦模式
- **Branch（分支）**：高亮当前节点的祖先与后代分支，其他节点淡化。
- **Path（路径）**：仅强调从根节点到当前节点的路径链路。
- **Highlight（高亮）**：保留全图可见，通过高亮边框突出当前分支。

### 临时总览
- 按住 `Alt`：临时显示被淡化内容，松开后恢复聚焦视图。

::: tip 使用建议
发散思考时优先用 `Branch`，做结构审校时切到 `Path`，做全局回顾时用 `Highlight`。
:::

## 🔄 格式兼容性

OneLook 支持与主流思维导图工具的数据互通：

### 支持的导入格式
- **XMind (.xmind)**：完整支持 XMind 8+ 格式，包括节点样式、备注、标签、优先级、进度等。
- **FreeMind (.mm)**：兼容 FreeMind XML 格式，保留节点结构、样式、备注和链接。
- **OPML (.opml)**：支持大纲处理标记语言，兼容多种大纲工具和 RSS 阅读器。
- **OneLook (.olook)**：原生格式，完整还原所有内容。
- **JSON (.json)**：通用数据格式。
- **Markdown (.md)**：从 Markdown 大纲创建思维导图。

### 支持的导出格式
- **第三方格式**：XMind (.xmind)、FreeMind (.mm)、OPML (.opml)
- **通用格式**：JSON (.json)、Markdown (.md)
- **图片格式**：PNG (.png)、SVG (.svg)
- **原生格式**：OneLook (.olook)

::: tip 智能转换
导入时会自动识别文件格式并转换为 OneLook 内部数据结构，导出时会根据目标格式进行相应的属性映射。
:::
