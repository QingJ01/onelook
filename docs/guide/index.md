# 快速开始

OneLook 的设计理念是“打开即用”。但为了让您更快进入心流状态，我们为您准备了几个典型场景的上手指引。

## 场景一：快速头脑风暴 (Brainstorming)
当灵感来袭时，速度最重要。

1. **打开 OneLook**：访问 [https://onelook.byebug.cn](https://onelook.byebug.cn)，您会直接看到一个中心主题。
2. **输入核心想法**：双击中心节点，输入您的主题（例如“产品发布会创意”）。
3. **疯狂发散**：
   - 按下 `Tab` 键，快速创建一个分支，输入一个想法。
   - 按下 `Enter` 键，立刻创建下一个平级想法。
   - **技巧**：不要停下来思考布局，先用键盘把所有关键词敲出来。
4. **整理归纳**：
   - 想法列完后，直接拖拽节点，将相关的点归类到一起。
   - 选中不靠谱的想法，按 `Delete` 删除。

## 场景二：整理读书笔记 (Note Taking)
经常需要记录层级分明的知识点。

1. **切换布局**：点击右上角设置，将布局切换为 **“树形结构 (Tree)”**，这更适合做笔记。
2. **录入内容**：
   - 利用 `Tab` 和 `Enter` 构建章节层级。
   - 选中节点打开右侧属性面板，在 **“备注”** 中粘贴大段的书摘。
3. **重点标记**：
   - 对重要的概念节点，在右侧改变其 **背景色** 或 **加粗** 文字。
   - 使用 **Markdown** 语法（如 `**重点**`）在节点内部直接高亮关键词。

## 场景三：学术/数学推导 (Academic)
OneLook 对理工科极其友好。

1. **输入公式**：在节点中直接输入 LaTeX，例如 `$\int_0^\infty e^{-x^2} dx$`。
2. **实时验证**：输入完按下 `Esc`，立即看到渲染后的数学公式。
3. **导出分享**：完成后，导出为 **SVG** 或 **PDF** (通过打印)，放入您的论文或作业中。

## 基础操作速查

| 动作 | 快捷键 | 鼠标操作 |
| :--- | :--- | :--- |
| **选中** | - | 点击节点 |
| **编辑** | `Space` / `F2` | 双击节点 |
| **新增子节点** | `Tab` | 选中节点右上角的 `+` |
| **新增兄弟节点** | `Enter` | - |
| **删除** | `Delete` | - |
| **移动画布** | `Space` + 拖拽 | 鼠标滚轮 |
| **缩放** | `Ctrl` + 滚轮 | 右下角 `+` / `-` |
