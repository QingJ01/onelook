# 快捷键大全

OneLook 专为键盘党设计，熟练使用快捷键可以成倍提升您的工作效率。

## 基础编辑

| 快捷键 | 动作 | 说明 |
| :--- | :--- | :--- |
| **Tab** | 添加子节点 | 选中节点后，快速创建下一级 |
| **Enter** | 添加兄弟节点 | 快速列出同级要点 |
| **Space** | 开始编辑 | 选中节点后按空格进入编辑模式 |
| **F2** | 开始编辑 | 同上 |
| **Esc** | 结束/取消 | 退出编辑模式，或取消选中的节点 |

## 编辑与调整

| 快捷键 | 动作 | 说明 |
| :--- | :--- | :--- |
| **Delete** | 删除节点 | 删除选中节点（包括子节点） |
| **Backspace** | 删除节点 | 同上 |
| **Ctrl + Z** | 撤销 | 撤销上一步操作 |
| **Ctrl + Y** | 重做 | 恢复被撤销的操作 |
| **Ctrl + Shift + Z** | 重做 | 同上 |

## 视图控制

| 快捷键 | 动作 | 说明 |
| :--- | :--- | :--- |
| **Space + 拖拽** | 移动画布 | 按住空格键不放，鼠标变为抓手 |
| **Ctrl + 滚轮** | 缩放视图 | 向上放大，向下缩小 |
| **Ctrl + +** | 放大 | |
| **Ctrl + -** | 缩小 | |
| **Ctrl + 0** | 重置视图 | 回到 100% 比例并居中 |
| **?** | 帮助面板 | 随时查看此列表 |

## 高级操作

| 快捷键 | 动作 | 说明 |
| :--- | :--- | :--- |
| **Ctrl + F** | 查找/替换 | 唤起全局搜索框 |
| **Ctrl + Click** | 多选 | 逐个点击添加选中节点 |
| **Shift + 拖拽** | 框选 | 在空白处拖拽矩形框批量选中 |
